##Javascript 화면제어
```
let boxTag = '';

for(let i=0; i<25; i++){
    boxTag += `<div class="bg${i%2===0? 2 : 1}"></div>`
};

document.getElementsByTagName(`section`)[0].innerHTML = boxTag
```

1. document : 브라우저 화면상의 정보에 접근
2. getElementsByTagName(`section`)[0] : 브라우저 화면 안에 있는 `section`이라는 태그에 접근; 화면상에 같은 태그가 여러개 있을 가능성이 있으므로 배열로 표시;
3. innerHTML = : 접속한 태그 안에 다음의 내용을 삽입함
