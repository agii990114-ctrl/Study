# Js 메소드

## 1.문자열 메소드
- 원본 보존 : 자바스크립트의 문자열은 **원시 타입(Primitive Type)**이므로, 메소드를 사용해도 원본 문자열은 변하지 않습니다. (항상 새로운 문자열을 반환합니다.)
- 메소드 체이닝 : 여러 메소드를 한 번에 연결해서 사용할 수 있습니다. ex) `str.trim().toLowerCase().includes('apple');`
- split()이나 slice()는 배열과 함께 코딩 테스트에서도 자주 쓰이는 아주 중요한 메소드

  
| 순번| 메소드 | 설명 | 비고 |
| :---: | :---: | :---: | :---: |
| 1 | length | 문자열 길이 |
| 2 | replace('바꾸고 싶은 문자', '바꿀 문자') | 일치하는 첫 번째 문자만 변경 |
| 3 | replaceAll('바꾸고 싶은 문자', '바꿀 문자') | 일치하는 모든 문자 변경 |
| 4 | substring(시작 인덱스, 끝 인덱스) | 특정 문자 추출 (끝 인덱스 포함 X) | 음수 사용 불가 |
| 5 | slice(시작 인덱스, 끝 인덱스) | 특정 문자 추출 (끝 인덱스 포함 X) | 음수 사용 가능 |
| 6 | split('기준 문자') | 기준 문자를 기준으로 좌우로 쪼개 배열로 표현 (기준 문자는 제외)<br>기준문자가 문자열의 끝일 경우 배열 마지막에 '' 인덱스가 생성 | let a="가나다" <hr> a.splite('') <hr> [가],[나],[다] |
| 7 | padStart(전체 문자 공간, '채울 문자') <hr> padStart(전체 문자 공간, '채울 문자') | 전체 문자 공간을 설정한 후 요소 외의 빈공간을 요소의 왼쪽에 생성하고 '채울 문자'로 채운다. <hr> 전체 문자 공간을 설정한 후 요소 외의 빈공간을 요소의 오른쪽에 생성하고 '채울 문자'로 채운다.  |
| 8 | includes('문자') | 문자열에서 문자의 존재 여부를 True/False로 출력  |
| 9 | indexOf('문자') | 문자열에서 문자 위치정보를 숫자로 출력 (시작: 0) |
| 10 | repeat(반복횟수, '문자') | 문자열에서 문자 위치정보를 숫자로 출력 (시작: 0) |
| 11 | toUpperCase('문자') | 문자를 대문자로 출력 |
| 12 | toLowerCase('문자') | 문자를 소문자로 출력 |
| 13 | trim() <hr> trimStart() <hr> trimEnd() | 요소의 바깥 양쪽 공백 제거 <hr> 요소의 바깥 왼쪽 공백 제거 <hr> 요소의 바깥 오른쪽 공백 제거 |
