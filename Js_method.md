# Js 메소드

## 1.문자열 메소드
| 메소드 | 설명 | 비고 |
| :--- | :--- | :--- |
| length | 문자열 길이 |
| replace(문자위치, '바꿀 문자') | 일치하는 첫 번째 문자만 변경 |
| replaceAll(문자위치, '바꿀 문자') | 일치하는 모든 문자 변경 |
| substring(시작 인덱스, 끝 인덱스) | 특정 문자 추출 (끝 인덱스 포함 X) |
| slice(시작 인덱스, 끝 인덱스) | 특정 문자 추출 (끝 인덱스 포함 X) |
| split('기준 문자') | 기준 문자를 기준으로 좌우로 쪼개 배열로 표현 (기준 문자는 제외) | let a="가나다"
|
| padStart/padEnd(전체 문자 길이, '채울 문자') | 전체 문자 공간에서 변수를 기준으로 변수 외의 빈 공간을 '채울 문자'로 채운다. |
| includes('문자') | 문자열에서 문자의 존재 여부를 True/False로 출력  |
| indexOf('문자') | 문자열에서 문자 위치정보를 숫자로 출력 (시작: 0) |
| repeat(반복횟수, '문자') | 문자열에서 문자 위치정보를 숫자로 출력 (시작: 0) |
