# Js 메소드

## 1.문자열 메소드

| 메소드 | 설명 | 비고 |
| :---: | :---: | :---: |
| length | 문자열 길이 |
| replace('바꾸고 싶은 문자', '바꿀 문자') | 일치하는 첫 번째 문자만 변경 |
| replaceAll('바꾸고 싶은 문자', '바꿀 문자') | 일치하는 모든 문자 변경 |
| substring(시작 인덱스, 끝 인덱스) | 특정 문자 추출 (끝 인덱스 포함 X) | 음수 사용 불가 |
| slice(시작 인덱스, 끝 인덱스) | 특정 문자 추출 (끝 인덱스 포함 X) | 음수 사용 가능 |
| split('기준 문자') | 기준 문자를 기준으로 좌우로 쪼개 배열로 표현 (기준 문자는 제외)<br>기준문자가 문자열의 끝일 경우 배열 마지막에 '' 인덱스가 생성 | let a="가나다" <hr> a.splite('') <hr> [가],[나],[다] |
| padStart(전체 문자 공간, '채울 문자') <hr> padStart(전체 문자 공간, '채울 문자') | 전체 문자 공간을 설정한 후 요소 외의 빈공간을 요소의 왼쪽에 생성하고 '채울 문자'로 채운다. <hr> 전체 문자 공간을 설정한 후 요소 외의 빈공간을 요소의 오른쪽에 생성하고 '채울 문자'로 채운다.  |
| includes('문자') | 문자열에서 문자의 존재 여부를 True/False로 출력  |
| indexOf('문자') | 문자열에서 문자 위치정보를 숫자로 출력 (시작: 0) |
| repeat(반복횟수, '문자') | 문자열에서 문자 위치정보를 숫자로 출력 (시작: 0) |
| toUpperCase('문자') | 문자를 대문자로 출력 |
| toLowerCase('문자') | 문자를 소문자로 출력 |
| trim() <hr> trimStart() <hr> trimEnd() | 요소의 바깥 양쪽 공백 제거 <hr> 요소의 바깥 왼쪽 공백 제거 <hr> 요소의 바깥 오른쪽 공백 제거 |
