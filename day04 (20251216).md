# 1. 복합 선택자

- A B  ==>  하위 선택자 : A 요소 내의 모든 B 요소를 선택
- A > B  ==>  자식 선택자 : A 요소 내의 직계 자식 B요소를 선택
- A + B  ==>  인접 형제 선택자 : A 요소 바로 다음에 오는 첫번째 B요소를 선택
- A ~ B  ==>  일반 형제 선택자 : A 요소 다음에 오는 모든 B요소를 선택

# 2. 속성 선택자

- = 전체 일치
- ^= 시작하는
- $= 끝나는
- *= 일부 문자 (단어 중 일부 문자)
- ~= 단어 일치

# 3. 가상 클래스 선택자

- a:link : 방문하지 않은 link에 효과
- a:visited : 방문한 link 효과
- a:hover : 마우스를 올렸을 때 효과
- a:active : 클릭했을 때
- a:focus : 키보드의 탭(tab)키를 눌러 지정됐을 때
