## React

1. 시작(터미널에 입력) // node 설치, cmd 터미널
- npx create-react-app "프로젝트명"
- npm create vite "프로젝트명"

2. index.html 내의 `<div id='root'></div>`에 내용이 첨가 됨

3. index.js(x) 파일에 다른 js(x) 파일들을 import를 사용 하면서 영역을 나눠 사용 가능

```
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
import App from './App.jsx'

createRoot(document.getElementById('root')).render(
  <StrictMode>
    <App />
  </StrictMode>,
)
```

5. 다른 js(x)파일을 내보낼 때는 export를 사용
```
const Page1 = () => <h1>상단 화면</h1>
const Page2 = () => <h1>하단 화면</h1>
const Page3 = (v,i) => <li key={i}>순서: {v}</li>

const Main = ({arr1}) => {
  
  return(
    <ul>
      {
        arr1?.map(Page3)
      }
    </ul>
  )
}


const App = () => {
  const arr = ['1번','2번','3번'];
  return (
    <>
     <Page1/>

     <Main arr1={arr}/>

     <Page2/>
    </>
  )
}

export default App

```
