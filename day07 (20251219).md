# Javascript
## 자료형 : 연산을 위하여 분류 (같은 타입끼리만 연산 가능 / number 5 + string '1' = '51' => 정상 X)
### Primitive Type (기본 타입)
```
- number : 숫자 (정수, 실수)
- bigInt : 정수 -(2의 53승 -1) ~ (2의 53승 -1)
- string : 문자열 (문자열에 포한된 number, bigInt, 특수기호도 string에 포함)
- boolean   : 논리값(true, false)
- undefined : 초기화되지 않은 상태, 변수 초기화에 사용 (return 값이 없음을 의미)
- null      : 객체 변수 초기화 사용
- NaN       : Not a Number
- symboll : 유일무이한 자료의 생성

( == : 값만을 비교 / === : 자료형까지 비교 )
```
### Reference Type (참조 타입)
```
- array
- function
- object
```
